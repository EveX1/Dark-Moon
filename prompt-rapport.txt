🎯 DÉTECTION AUTOMATIQUE OBLIGATOIRE (DVGA)
──────────────────────────────────────────────

Tu dois détecter *dans n’importe quel fichier* :

1) GRAPHQL INTROSPECTION
   Motifs : "__schema", "__type", "Mutation", "Query".
   → Vulnérabilité : 🟧 "GraphQL introspection active"

2) DB LOOT / USERS / PASSWORDS
   Motifs : "users", "username", "password", "token".
   → Vulnérabilité : 🟥 "Fuite comptes / mots de passe"

3) EXFILTRATION PASTES & METADATA
   Motifs : "pastes", "content", "ipAddr", "userAgent".
   → Vulnérabilité : 🟧 "Fuite données sensibles (pastes)"

4) AUDIT LEAK
   Motifs : "audits", "gqlquery", "gqloperation".
   → Vulnérabilité : 🟧 "Fuite d'audits internes"

5) DEBUG ENDPOINTS
   Motifs : "systemDebug", "systemDiagnostics", "systemHealth", "systemUpdate".
   → Vulnérabilité : 🟧 "Endpoints système exposés"

6) XSS CONFIRMÉE
   Motifs : "<script", "onerror=", "<img src".
   → Vulnérabilité : 🟥 "XSS stockée confirmée"

7) NUCLEI GRAPHQL
   Motifs : "graphql-array-batching", "graphql-alias-batching", "graphql-detect", "SUCCESS:true".
   → Vulnérabilité : 🟧 "Faiblesses GraphQL supplémentaires"

8) ZAP / SEC HEADERS
   Motifs : Absence CSP, XFO, nosniff, CORP, fetch-*.
   → Vulnérabilités : 🟨 ou 🟧 selon criticité.

Si un pattern est détecté → tu crées une vulnérabilité.
Si plusieurs patterns similaires → tu fusionnes.
